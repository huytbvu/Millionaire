Millionaire 2012
by Huy Vu

-------------------------------------------------
game reflect the "Who wants to be a millionaire" TV show

15 questions, easy to hard level (more questions to be updated)

4 choices for each questions

wrong answer, stop the game and earn 50% score

decide to stop, earn 70% score

answer MUST be clicked on the NUMBER (1,2,3,4)

3 helping options: 50/50, Polling, Change (each can only used once)

50/50: program gives 2 INCORRECT answers

Polling: tell user what Most audience think (audience have 85% accuracy)

Change: change question
